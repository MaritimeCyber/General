# Asset

This directory contains all static resources and knowledge assets maintained by the Maritime 4.0 Crew.

## Structure

```
Asset/
├── Books/          # Reviews and summaries of maritime cyber security books
├── Paper/          # Research papers and technical reports
├── Compliances/    # Cyber compliance documents and class society guidelines
├── Project/        # Project deliverables, tools, and templates
└── img/
    └── member/     # Team member profile images
```

## Purpose

All materials stored here are intended to support research and practice in maritime cyber security. Content is organized by type to allow contributors and readers to navigate quickly to the area most relevant to their work.

> For an overview of the entire repository, see the [root README](../README.md).
