# Blog — Maritime 4.0 Blog Assets

This directory contains HTML templates and widgets used in the
[Maritime 4.0: Innovation Driven by AI, Data, and Cyber Security](https://shippauljobs.blogspot.com/) blog.

## Structure

```
Blog/
├── SideBar/                              # Sidebar widgets for the blog
│   ├── Find Us on GitHub.html            # GitHub repository link widget
│   ├── Maritime Cyber ​​Intelligence.html  # Blog introduction widget
│   ├── Who We Are.html                   # Team introduction widget
│   └── shipjobsIcon.html                 # Blog icon/logo widget
└── contents/                             # Article layout templates
    ├── Article Template1.html            # Standard article template
    └── Article Template2.html            # Alternative article template
```

## Purpose

- **SideBar/** — Ready-to-paste HTML gadgets for the Blogger sidebar. Each widget is self-contained and can be added via the Blogger layout editor.
- **contents/** — Article HTML templates that provide consistent formatting for blog posts covering maritime cyber security topics (SCARP, IACS UR E26/E27, IMO guidelines, OT/IT security, etc.).

## Usage

1. Open the target `.html` file.
2. Copy the full content.
3. In Blogger, navigate to **Layout → Add a Gadget → HTML/JavaScript** (for sidebar widgets) or paste into the post editor in HTML view (for article templates).
4. Save and preview.
